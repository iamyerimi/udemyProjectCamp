1. HTML 제목은 <h1> ~ <h6> 태그로 정의됩니다. <h1>은 가장 중요한 제목을, <h6>는 가장 중요하지 않은 제목을 정의합니다:

This is heading 1

This is heading 2

This is heading 3

2. 단락은 <p>태그로 정의됩니다.

This is a paragraph.

This is another paragragh.

3. 링크는 태그로 정의됩니다.

This is naver link.

4. 이미지는 태그로 정의됩니다. 소스 파일(src), 대체텍스트(alt), 너비 및 높이가 속성으로 제공됩니다:
